\chapter{EPS Línea de Tiempo de Tareas}

Esta es la ruta crítica técnica. El avance a la siguiente fase está estrictamente condicionado a la aprobación del entregable de la fase anterior:

\begin{enumerate}
    \item \textbf{Definición de Requisitos y Restricciones:} Establecer la base del diseño.
    \begin{itemize}
        \item \textit{Hito:} Aprobación del \textbf{SRD} (Líderes) y del \textbf{Mechanical ICD} (Agentes Dobles).
    \end{itemize}
    \item \textbf{Presupuesto de Potencia:} Cuantificar la demanda de energía.
    \begin{itemize}
        \item \textit{Hito:} Cierre del \textbf{Power Budget Master} (Líderes y SART). \textbf{ESTO DEFINE LA NECESIDAD REAL.}
    \end{itemize}
    \item \textbf{Selección de Componentes:} Buscar la solución comercial al problema planteado.
    \begin{itemize}
        \item \textit{Hito:} Entrega del \textbf{Trade-Off Report} y aprobación del \textbf{BOM Preliminar} (Asignatura).
    \end{itemize}
    \item \textbf{Simulación Física (Fase 1 - Teórica):} Validar la topología matemática.
    \begin{itemize}
        \item \textit{Hito:} Entrega del \textbf{PSIM Simulation Report v1.0} con componentes ideales (Asignatura).
    \end{itemize}
    \item \textbf{Desarrollo Lógico y Simulación Física (Fase 2 - Real):} Definir la inteligencia y probar la robustez del hardware frente a fallos.
    \begin{itemize}
        \item \textit{Hito:} Entrega del \textbf{State Machine Diagram} (SART) y del \textbf{PSIM Simulation Report v2.0} (Asignatura).
    \end{itemize}
    \item \textbf{Definición de Interfaces OBC:} Conectar el cerebro con el resto del satélite.
    \begin{itemize}
        \item \textit{Hito:} Publicación del \textbf{Interface Control Document - ICD} (SART).
    \end{itemize}
    \item \textbf{Captura de Esquemáticos y Revisión Crítica:} Diseño formal en ECAD y auditoría de seguridad.
    \begin{itemize}
        \item \textit{Hito:} Ejecución de la revisión y firma del \textbf{CDR Acta} (Todos).
    \end{itemize}
    \item \textbf{Cierre de Diseño:} Generación de archivos de fabricación y memorias.
    \begin{itemize}
        \item \textit{Hito:} Entrega del \textbf{Schematics Package} (Asignatura).
    \end{itemize}
\end{enumerate}
